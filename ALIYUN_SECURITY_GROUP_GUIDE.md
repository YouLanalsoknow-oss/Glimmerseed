
# 阿里云安全组端口配置指南

## 需要开放的端口

| 端口 | 协议 | 授权对象 | 用途 |
|------|------|---------|------|
| 3389 | TCP | 0.0.0.0/0 | 远程桌面（RDP）- 已开放 |
| 8080 | TCP | 0.0.0.0/0 | 后端 API 服务 |
| 3306 | TCP | 127.0.0.1 | MySQL（仅本地访问）|

---

## 配置步骤

### 第一步：登录阿里云控制台

1. 访问：https://ecs.console.aliyun.com/
2. 登录你的阿里云账号

### 第二步：找到你的 ECS 实例

1. 在左侧菜单选择「实例与镜像」→「实例」
2. 找到你的服务器实例（IP: 8.134.80.158）
3. 点击实例 ID 或「管理」

### 第三步：进入安全组配置

1. 在实例详情页面，找到「安全组」选项卡
2. 点击「配置规则」

### 第四步：添加入方向规则

点击「手动添加」，按以下表格添加规则：

#### 规则 1：开放 8080 端口（后端 API）

| 配置项 | 值 |
|--------|-----|
| 授权策略 | 允许 |
| 优先级 | 1 |
| 协议类型 | 自定义 TCP |
| 端口范围 | 8080/8080 |
| 授权对象 | 0.0.0.0/0 |
| 描述 | Backend API |

#### 规则 2：MySQL（可选，仅本地）

| 配置项 | 值 |
|--------|-----|
| 授权策略 | 允许 |
| 优先级 | 2 |
| 协议类型 | 自定义 TCP |
| 端口范围 | 3306/3306 |
| 授权对象 | 127.0.0.1/32 |
| 描述 | MySQL Local |

---

## 快速检查清单

配置完成后，再次运行诊断确认端口开放：

```powershell
# 在本地 PowerShell 运行
Test-NetConnection -ComputerName 8.134.80.158 -Port 8080
```

---

## 注意事项

⚠️ **安全提醒**：
- 不要将 MySQL (3306) 端口开放到公网 (0.0.0.0/0)
- 生产环境建议限制 8080 端口的访问源 IP
- 定期检查安全组规则

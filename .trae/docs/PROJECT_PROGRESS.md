# Glimmerseed项目发展全记录

## 1.项目基础信息

### 项目名称
Glimmerseed - 自研2D骨骼动画编辑器+安卓悬浮桌宠+本地离线AI对话工具

### 产品定位
一个功能完整的自研2D骨骼动画编辑器、动画制作工具，配合悬浮窗桌宠展示与AI对话功能的Android应用。全项目Skeleton骨骼、IK、PoseSnapshot姿势系统、RenderEngine渲染引擎全部自研，仅做第三方格式兼容：支持导入/导出Live2D、Spine文件。

### 商业化方案
- 爱发电订阅模式
- 自研编辑器内核、.glimmerseed工程格式全功能永久免费
- Live2D/Spine第三方商用导出格式划入订阅增值权益
- 云同步功能（付费）
- 高级模型导出格式支持

### 全技术栈
- **前端/客户端**: Kotlin/Android, Jetpack Compose, OpenGL ES
- **动画编辑器**: 自定义EditorCore模块，PoseSnapshot姿势快照系统
- **渲染引擎**: 自定义RenderEngine模块
- **桌宠悬浮窗**: FloatingPreviewBase模块
- **后端服务**: Java 21 + Spring Boot (原始)，后有FastAPI版本
- **AI集成**: Qwen-VL, MiniCPM视觉大模型
- **向量数据库**: PGVector (PostgreSQL)
- **本地部署**: Python + FastAPI + PostgreSQL
- **Web端预览**: React + TypeScript + Vite

### 硬件部署环境
- **本地开发**: 16G内存，256G硬盘
- **服务器部署**: 阿里云ECS，Windows Server
- **最低配置**: 8G内存，128G存储

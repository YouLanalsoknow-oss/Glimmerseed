
# 服务器部署指南

## 第一步：连接到服务器

1. 双击运行 `connect-rdp.bat` 文件，或者：
   - 按 `Win + R`
   - 输入：`mstsc /v:8.134.80.158`
   - 回车

2. 输入你的服务器用户名和密码登录

## 第二步：检查服务器环境

在服务器上运行 `list-files.bat`，查看服务器上已安装的软件

## 第三步：将文件传输到服务器

### 方法 A：使用远程桌面的本地资源共享（推荐）

1. 在远程桌面连接窗口，点击「显示选项」
2. 切换到「本地资源」选项卡
3. 点击「详细信息」
4. 勾选「驱动器」（勾选你的 C 盘或 D 盘）
5. 连接后，在服务器的「此电脑」中可以看到你的本地磁盘
6. 直接复制文件

### 方法 B：使用云盘/网盘上传下载

## 需要复制到服务器的文件

把以下文件/文件夹复制到服务器的 `C:\Glimmerseed\` 目录：

```
C:\Glimmerseed\
├── list-files.bat
├── backend\              (整个文件夹)
│   ├── pom.xml
│   ├── init_db.sql
│   ├── build.bat
│   ├── start.bat
│   └── src\
└── ...
```

## 第四步：配置安全组（阿里云控制台）

登录阿里云控制台 → 云服务器 ECS → 实例 → 安全组 → 配置规则

需要添加以下入方向规则：

| 端口范围 | 授权对象 | 说明 |
|---------|---------|------|
| 8080    | 0.0.0.0/0 | 后端 API |
| 3306    | 127.0.0.1 | MySQL (仅本地) |

## 第五步：在服务器上安装软件

需要检查并安装：

1. Java 17+ (JDK)
2. Maven 3.8+
3. MySQL 8.0+

## 第六步：运行后端

在服务器上打开命令行，进入 `C:\Glimmerseed\backend` 目录：

```cmd
cd C:\Glimmerseed\backend
build.bat
start.bat
```
